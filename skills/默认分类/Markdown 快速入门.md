# Markdown 快速入门

欢迎使用 **私人 Skill MD 管理平台**！

这是一个默认的技能包，用来展示平台对于 Markdown 的精美排版和高逼格设计。

## 平台特色功能

- 📂 **本地绝对路径热重载**：在右上角设置中指定任何电脑目录，一键热切知识库。
- 🎨 **包豪斯极简画廊设计**：极致优雅的白底黑字、精细线条骨架与轻量扁平悬浮微动效。
- 📝 **双栏实时编辑预览**：在网页端随时添加、修改你的技能 MD，直接同步本地文件。
- 🔗 **全方位分享机制**：支持生成精美海报图片、导出自带样式的单文件 HTML、局域网共享。

### 快捷操作指南

点击卡片底部的 **👁️** 图标进入沉浸式阅读模式。在阅读器右上角：
1. 点击 **编辑此篇** 可以直接进行双栏修改。
2. 点击 **🔗 分享 / 导出** 可以生成图片海报或导出为精美的独立 HTML 页面。
3. 点击 **📥 下载 md** 获取原始文件。
